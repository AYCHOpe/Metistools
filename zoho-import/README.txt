The class eu.europeana.metis.zoho.OrganizationImporter contains a main method. The main method accepts the optional parameter for importing only organizations created or modified after the provided date formatted as ZOHO_TIME_FORMAT = "yyyy-MM-dd hh:mm:ss" (e.g. "2018-03-12 10:07:22")

The property file zoho_import.properties is expected to be in the root of the classpath (i.e. resource directory, see 
example file).
